<p>We create a vector, call the length method and print the result:</p>
</div>

{% highlight python %}

u = la.Mat([[1, 1, 3]])

u_length = u.length()

print(u_length)

{% endhighlight %}

Outputs:

{% highlight shell %}

>>> print(u_length)
3.3166247903554

{% endhighlight %}
